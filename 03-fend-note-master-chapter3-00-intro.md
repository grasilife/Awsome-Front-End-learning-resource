DOM 编程就是使用 W3C 定义的 API (Application Program Interface)
来操作 HTML 文档
（此处不局限于 HTML，亦可操作 XHTML、XML  等），使用户可以与进行页面交互。
你需要了解节点、属性、样式等基本 DOM 操作，DOM 事件模型，数据存储
(Cookie、Storage) 与数据通信 (Ajax) ，JavaScript 动画，音频、视频、Canvas
等 HTML5 特性，表单、列表操作。
